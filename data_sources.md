# Dataset Source

- Dataset: Campus Recruitment CSV
- Platform: Hugging Face Datasets
- Repository: https://huggingface.co/datasets/Krooz/Campus_Recruitment_CSV
- License: CC0-1.0
- Files used:
  - `raw_datasets/train.csv`
  - `raw_datasets/test.csv`
  - `raw_datasets/validation.csv`

## Why This Dataset Fits The Project

This dataset is suitable for student employability prediction because it contains academic and skill-related fields such as:

- CGPA
- Internships
- Projects
- Workshops and certifications
- Aptitude test score
- Soft skills rating
- Extracurricular activities
- Placement training
- SSC and HSC marks
- Placement status

The target variable is `PlacementStatus`, which makes the dataset suitable for a supervised Machine Learning classification model.
